Query Tips

  • Use SELECT * to see all columns.
  • Filter rows with WHERE clause.
  • Combine tables using JOIN.
  • Order results with ORDER BY.
  • Aggregate data with GROUP BY.
  • Try COUNT(), SUM(), AVG().
  • Case sensitive for table/column names in AlaSQL.
